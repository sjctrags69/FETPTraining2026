How to start Maven Project
1. Go to File > New > Other...
2. Maven > Maven Project
3. Check "Create a simple project..."
4. type "org.fujitsu.training.codes" in Group Id: 
5. type "mymaven" in Artifact Id:
6. type "1.0" in Version
7. choose "jar" in Packaging
8. click Next > Finish

=====
In pom.xml, if line 1 <project xmlns....> is showing error
1. Go to Window > Preferences...
2. Click > XML (Wild Web Developer) 
3. Check if unchecked "Download external resources like referenced DTD, XSD"
4. Click Apply > Apply and Close 

=====
To change or update JRE System Library [JavaSE - version]
1. Go to root folder (folder with Artifact Id)
2. Right-click > 'Properties'
3. Double-Click 'Project Facets'
4. Check 'Java' 
5. Choose Version from the dropdown (e.g. 24)
6. Click 'Apply' > 'Apply and Close'

Result:
JRE System Library [JavaSE-24]

pom.xml

<project xmlns="http://maven.apache.org/POM/4.0.0" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 https://maven.apache.org/xsd/maven-4.0.0.xsd">
  <modelVersion>4.0.0</modelVersion>
  <groupId>org.fujitsu.training.codes</groupId>
  <artifactId>mymaven2.0</artifactId>
  <version>2.0</version>
  <name>MyMavenPractice</name>
  <description>Practice starting a maven project</description>
  <properties>
  <maven.compiler.source>24</maven.compiler.source>
  <maven.compiler.target>24</maven.compiler.target>
  <project.build.sourceEncoding>UTF-8</project.build.sourceEncoding>
  <java.version>24</java.version>
  <source.encoding>UTF-8</source.encoding>
  </properties>
  <build>
  	<plugins>
  		<plugin>
  			<groupId>org.apache.maven.plugins</groupId>
  			<artifactId>maven-compiler-plugin</artifactId>
			<version>3.15.0</version>
			<configuration>
				<source>${java.version}</source>
				<target>${java.version}</target>
				<encoding>${source.encoding}</encoding>
			</configuration>
  		</plugin>
  	</plugins>
  </build>
</project>
